# 🚀 Smart Code Analyzer

A full-stack web application that analyzes code snippets and estimates their time complexity using pattern-based static analysis.

🔗 **Live Demo:** https://code-analyzer-1-66h7.onrender.com/

---

## 🔥 Features

* 🔍 Detects time complexity:

  * O(1)
  * O(n)
  * O(n²)
  * O(n³)
  * O(log n)
  * O(n log n)

* 💡 Provides optimization suggestions

* 🎨 Modern dark UI with color-coded results

* 📋 Copy-to-clipboard feature

* ⚡ Real-time code analysis

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Flask (Python)
* **Deployment:** Render

## ⚙️ How It Works

1. User enters code snippet
2. System analyzes:

   * Loops (`for`, `while`)
   * Nested structures
   * Recursion
   * Sorting patterns
   * Binary search logic
3. Returns:

   * Estimated time complexity
   * Optimization suggestion
